### File Formats
- [ ] Project folder is zipped as **.zip** (NOT .rar, .7z, .tar, etc.)
- [ ] Report is in **.docx** format (NOT .pdf, .pages, .odt, .doc, etc.)
- [ ] Links file is in **.txt** format (NOT .doc, .pdf, etc.)

### File Naming
- [ ] Project folder: `ProjectName_GroupName_FinalProject.zip`
- [ ] Report: `ProjectName_GroupName_Report.docx`
- [ ] Links: `ProjectName_GroupName_Links.txt`
- [ ] Example: `PortfolioHub_SE2435_FinalProject.zip`

### Required Files
- [ ] **Project Folder (.zip)** containing:
  - [ ] All HTML files (index.html, wezeso.html, alikhan.html, amirkhan.html, sardor.html, login.html)
  - [ ] All CSS files (styles.css)
  - [ ] All JavaScript files (hub.js, script.js, auth.js, portfolio-lookup.js)
  - [ ] All image files (if any)
  - [ ] All other required files
  
- [ ] **Report (.docx)** containing:
  - [ ] Project Title and Team Members
  - [ ] Project Goal and Description
  - [ ] Work Division (who did what)
  - [ ] Detailed Explanation of Website
  - [ ] Screenshots
  
- [ ] **Links File (.txt)** containing:
  - [ ] GitHub Repository URL
  - [ ] Live Website URL

### Individual Submission
- [ ] **EACH team member** submits all 3 files individually
- [ ] Even though it's a team project, EVERY student must upload
- [ ] Do NOT assume one person's submission covers everyone

### Submission Location
- [ ] Files are submitted in **Moodle IN FINAL EXAM FOLDER**
- [ ] NOT in regular assignment folder
- [ ] NOT via email or other means
